🎯 프로젝트 개요

실무 중심의 프로젝트 수행 능력을 기르는 차세대 학습 관리 시스템입니다.

  • 프로젝트 기반 학습(PBT) 관리
  • 실시간 코드 실행 및 평가
  • 팀 협업 및 토론 기능
  • 개인별 학습 진도 추적

🛠️ 기술 스택

최신 웹 기술을 활용한 현대적인 플랫폼입니다.

  • Backend: Node.js + Express
  • Database: MySQL + Sequelize
  • Frontend: React + Material-UI
  • 실시간: Socket.io
